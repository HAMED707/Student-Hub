# Student-Hub

## 🚧 Project Status: In Progress

This project is currently under active development and is not yet complete.

## 📋 Description

Accommodation Finder is a comprehensive web application designed to help users discover, list, and book accommodation options. The platform connects property owners with potential renters, providing an intuitive interface for browsing available properties with advanced search and filtering capabilities including geolocation-based queries.

## 🛠️ Tech Stack

### Backend
- Python 3.x
- Django
- Django REST Framework
- PostgreSQL
- PostGIS (for geographical queries)
- Celery + Redis (background tasks)
- Sentry (error monitoring)

### Frontend
- React.js
- JavaScript (ES6+)
- Vite
- Tailwind CSS
- React Router DOM
- Axios
- Leaflet & React Leaflet (Maps Integration)
- JWT Authentication
- Responsive UI Design
- Reusable Components Architecture
